# sd-data-visualization by PTSD

## How to run on your local machine
1. Clone this repo

### How to get the data and load it on the database
1. `cd` into the repo
1. Run `source get_raw_data.sh`
1. Then `cd app/`
1. Run `python3 load_data.py`
